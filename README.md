## How I Build

I prefer pragmatic engineering with strong taste.

Good work should be:

- clear before it becomes complex
- visually intentional
- accessible and responsive
- type-safe and maintainable
- emotionally resonant without becoming decorative noise
- built on reusable systems where reuse actually helps

I like interfaces that feel calm, cinematic, precise, and alive.
