## Philosophy

I prefer pragmatic systems with strong visual taste.

Good frontend work should be:

- clear before complex
- expressive without becoming noisy
- accessible and responsive
- maintainable at scale
- grounded in real usage, not trends

I like interfaces that feel calm, cinematic, tactile, and precise.
